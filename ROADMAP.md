# Roadmap

## Alpha Release (Current)
- Basic layer implementations (Dense, Conv2D, Dropout, BatchNormalization, Flatten)
- Core callbacks (EarlyStopping, ModelCheckpoint, CSVLogger, ReduceLROnPlateau)
- SGD and Adam optimizers
- L2 regularization
- Training with mini-batch support

## Beta Release
- [ ] L1 regularizer
- [ ] More initializers
- [ ] Model summary method
- [ ] Gradient clipping
- [ ] Better serialization format (JSON schema)

## Future
- [ ] Recurrent layers (RNN, LSTM, GRU)
- [ ] Pooling layers (MaxPool, AvgPool)
- [ ] Embedding layer for text
- [ ] Stateful optimizers (save/load optimizer state)
- [ ] Learning rate schedulers
- [ ] Model evaluation methods
- [ ] Data pipeline utilities
